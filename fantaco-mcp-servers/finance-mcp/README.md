## Setup
```
python3.14 -m venv .venv
source ./.venv/bin/activate
pip install -r requirements.txt
```

## Running the Server

```bash
python finance-api-mcp-server.py
```

The server will start on the configured host and port (default: http://0.0.0.0:9001).
